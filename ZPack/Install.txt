ZPack Bedrock Edition Installation Guide

Cut/Copy the unzipped "ZPack.Vanilla-BE" folder

Open File Explorer, and in the address bar, enter %AppData%

Navigate to; > Roaming > Minecraft Bedrock > Users > Shared > games > com.mojang > resource_packs

Paste the "ZPack.Vanilla-BE" folder & restart Minecraft: Bedrock Edition

All done, now your vanilla is better than ever!

ZPack Java Edition Installation Guide

Cut/Copy the unzipped "ZZPack.Vanilla-JE" folder

Open Minecraft: Java Edition with your preferred launcher

In Minecraft, select; Options, Resource Packs, "Open Pack Folder"

Inside the file browser window, paste the "ZPack.Vanilla-JE" folder.

All done, now your vanilla is better than ever!
