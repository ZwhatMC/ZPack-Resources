Bedrock Edition Installation Guide

Cut/Copy the "ZPack Bedrock 26" folder

Open File Explorer, and in the address bar, enter %AppData%

Navigate to; > Roaming > Minecraft Bedrock > Users > Shared > games > com.mojang > resource_packs

Paste the "ZPack Bedrock 26" folder & restart Minecraft: Bedrock Edition

All done, now your vanilla is better than ever!

--------------------------------------------------

Java Edition Installation Guide

Cut/Copy the "ZPack Java 26" folder

Open Minecraft: Java Edition with your preferred launcher

In Minecraft, select; Options, Resource Packs, "Open Pack Folder"

Inside the file browser window, paste the "ZPack Java 26" folder.

All done, now your vanilla is better than ever!
